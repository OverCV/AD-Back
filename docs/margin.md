# Marginalizaci√≥n

Tenemos S2S:

[[c11, c12, ..., c1N],
 [c21, c22, ..., c2N],
 ...
 [cM1, cM2, ..., cMN]]

Hay que llevar de S2S a S2C para no tener que marginalizar por cada elemento.


Tenemos S2C:

[[A11, B11, ..., CM1],
 [A12, B12, ..., CM2],
 ...
 [A1N, B1N, ..., CMN]]

 Hay que llevar de S2C a S2P para poder realizar los productos tensor.

Tenemos S2P:

[[A11, A12, B13, B14, ..., Z1N],
 [A21, A22, B23, B24, ..., Z2N],
 ...
 [AM1, AM2, BM3, BM4, ..., ZMN]]
